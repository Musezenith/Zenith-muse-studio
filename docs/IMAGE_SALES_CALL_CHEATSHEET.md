# Musezenith Image Product - Live Sales Call Cheat Sheet (One Page)

## 1) 20-sec pitch
“Musezenith turns image generation into a full decision workflow: brief -> variants -> compare -> final selection -> export bundle. Teams move faster from concept to approved visual with less prompt chaos and cleaner handoff.”

## 2) Fast fit check (60 sec)
Strong-fit signals:
- Runs visual concept iterations weekly
- Team reviews/approves outputs (not solo-only)
- Current pain in compare/decision/handoff, not just generation

Weak-fit signals:
- Very low image iteration volume
- No real review workflow
- Looking only for a single “one-click image generator”

## 3) Core discovery questions (pick 4-6)
1. “How often do you run image concept cycles?”
2. “Where do you lose most time: prompting, comparing, or approvals?”
3. “How do you choose final outputs today?”
4. “How do you share rationale/history with internal teams or clients?”
5. “What causes most revision loops?”
6. “If this works, what should improve in 2 weeks?”

## 4) Live demo flow (real app flow, 8-12 min)
1. PromptLab:
   - Enter brief, set preset/model/variants/seed policy
2. Generate:
   - Click `Generate Prompt + Images`
   - Mention run states + retry/cancel
3. Review tabs:
   - Positive / Negative / Params / QC / JSON / Score
4. Show generated variants
5. Archive:
   - Find run, open detail
6. Compare:
   - Send options to Compare Left/Right
7. Final selection:
   - Mark run final + image final(s)
8. Re-run:
   - Trigger `Re-run` from archive
9. Export:
   - `Export bundle` from detail drawer

## 5) Strongest value points (say these)
- “Not just generation: decision workflow + traceable handoff”
- “Compare + final selection reduces subjective back-and-forth”
- “Archive + re-run makes iteration reproducible”
- “Remote-first assets with fallback behavior keep workflow resilient”
- “Export bundle gives clean artifact for review/support”

## 6) Common objections (short responses)
- “We already use image generators”
  - “This layer solves compare/finalize/handoff after generation.”
- “What if generation fails?”
  - “Run states + retry/cancel are built in.”
- “What if asset links break?”
  - “Remote-first with fallback behavior; workflow remains usable.”
- “How do we present to clients?”
  - “Use export bundle with prompt/result/metadata/assets references.”

## 7) Call close (offer one clear next step)
Primary CTA:
- “Let’s run a 2-week pilot on one real campaign brief.”

Alternative CTA options:
1. “Book a pilot kickoff session (45 min).”
2. “Do one guided live brief run this week.”
3. “Align on success criteria + owner, then start pilot.”

Close line:
- “If we reduce your concept-to-approval friction in 2 weeks, are you open to moving this into your standard workflow?”
